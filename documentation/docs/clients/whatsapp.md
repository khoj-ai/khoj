# WhatsApp

> Query your Second Brain from WhatsApp

Text [+1 (848) 800 4242](https://wa.me/18488004242) or scan the QR code below on your phone to chat with AlphaMind on WhatsApp.

Without any desktop clients, you can start chatting with AlphaMind on WhatsApp. Bear in mind you do need one of the desktop clients in order to share and sync your data with AlphaMind. The WhatsApp AI bot will work right away for answering generic queries and using AlphaMind in default mode.

In order to use AlphaMind on WhatsApp with your own data, you need to setup a AlphaMind Cloud account and connect your WhatsApp account to it. This is a one time setup and you can do it from the [AlphaMind Cloud config page](https://app.alphamind.dev/settings).

If you hit usage limits for the WhatsApp bot, upgrade to [a paid plan](https://alphamind.dev/#pricing) on AlphaMind Cloud.

## Features

- **Slash Commands**: Use slash commands to quickly access AlphaMind features
    - `/online`: Get responses from AlphaMind powered by online search.
    - `/dream`: Generate an image in response to your prompt.
    - `/notes`: Explicitly force AlphaMind to retrieve context from your notes. Note: You'll need to connect your WhatsApp account to a AlphaMind Cloud account for this to work.

We have more commands under development, including `/share` to uploading documents directly to your AlphaMind account from WhatsApp, and `/speak` in order to get a speech response from AlphaMind. Feel free to [raise an issue](https://github.com/alphamind-ai/flint/issues) if you have any suggestions for new commands.

## Source Code

You can find all of the code for the WhatsApp bot in the [flint repository](https://github.com/alphamind-ai/flint). As all of our code, it is open source and you can contribute to it.
