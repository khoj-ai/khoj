# Desktop

> Upload your knowledge base to AlphaMind and chat with your whole corpus

## Companion App

Share your files, folders with AlphaMind using the app.
AlphaMind will keep these files in sync to provide contextual responses when you search or chat.

## Setup
:::info[Self Hosting]
If you are self-hosting the AlphaMind server, update the *Settings* page on the AlphaMind Desktop app to:
- Set the `AlphaMind URL` field to your AlphaMind server URL. By default, use `http://127.0.0.1:42110`.
- Do not set the `AlphaMind API Key` field if your AlphaMind server runs in anonymous mode. For example, `alphamind --anonymous-mode`
:::


1. Install the [AlphaMind Desktop app](https://alphamind.dev/downloads) for your OS
2. Generate an API key on the [AlphaMind Web App](https://app.alphamind.dev/settings#clients)
3. Set your AlphaMind API Key on the *Settings* page of the AlphaMind Desktop app
4. [Optional] Add any files, folders you'd like AlphaMind to be aware of on the *Settings* page and Click *Save*.
   These files and folders will be automatically kept in sync for you

# Main App

You can also install the AlphaMind application on your desktop as a progressive web app.

1. Open the [AlphaMind Web App](https://app.alphamind.dev) in Chrome.
2. Click on the install button in the address bar to install the app. You must be logged into your Chrome browser for this to work.

![progressive web app install icon](/img/pwa_install_desktop.png)

Alternatively, you can also install using this route:

1. Open the three-dot menu in the top right corner of the browser.
2. Go to 'Cast, Save, and Share' option.
3. Click on the "Open in AlphaMind" option.


![progressive web app install route](/img/chrome_pwa_alt.png)
